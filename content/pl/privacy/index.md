#### Polityka prywatności

Portal sysadmin.info.pl nie przetwarza, nie udostępnia ani nie wykorzystuje w&nbsp;celach marketingowych danych przekazywanych przez Użytkowników portalu. Możecie być pewni że&nbsp;w&nbsp;przyszłości także nie będziemy przetwarzać, udostępniać ani wykorzystywać waszych danych w&nbsp;jakichkolwiek innych celach, niż zwykłe zbieranie statystyk.

Portal sysadmin.info.pl nie ponosi odpowiedzialności za&nbsp;treści i&nbsp;opinie wyrażane przez użytkowników w&nbsp;komentarzach lub opisach. Są one wyłącznie własnością użytkowników, od&nbsp;których pochodzą.

#### Hugo a ogólne rozporządzenie o ochronie danych osobowych (GDPR)

Strona sysadmin.info.pl jest generowana w generatorze statycznym Hugo.

Więcej informacji tutaj: https://gohugo.io/about/hugo-and-gdpr/

Poniżej znajdują się wszystkie ustawienia prywatności oraz ich domyślna wartość. Ustawienia te zostały umieszczone w pliku config.toml strony sysadmin.info.pl

```
[privacy]
  [privacy.disqus]
    disable = true
  [privacy.googleAnalytics]
    anonymizeIP = true
    disable = true
    respectDoNotTrack = true
    useSessionStorage = true
  [privacy.instagram]
    disable = true
    simple = true
  [privacy.twitter]
    disable = true
    enableDNT = true
    simple = true
  [privacy.vimeo]
    disable = true
    enableDNT = true
    simple = true
  [privacy.youtube]
    disable = false
    privacyEnhanced = false
```

#### Adresy E-mail

Adresy E-mail podawane przez Użytkowników przy dodawaniu komentarzy, oraz&nbsp;rejestracji na&nbsp;forum lub portalu, będzie używany jedynie w&nbsp;celu przypomnienia hasła Użytkownika, oraz&nbsp;wysłania Newsletter’a – jeśli Użytkownik wyraził taką chęć. Adresy E-mail Użytkowników nie są i&nbsp;nie będą nigdzie publikowane, przetwarzane, sprzedawane, udostępniane za&nbsp;opłatą – w&nbsp;związku z&nbsp;czym nie musicie się obawiać dodawania do&nbsp;żadnych list mailingowych, otrzymywania sponsorowanych maili reklamowych, i&nbsp;wszelkiego innego niechcianego spamu.

#### Gromadzenie danych nie-osobowych

Podczas użytkowania portalu sysadmin.info.pl, zbierane są dane nie-osobowe które przeglądarki udostępniają, czyli:

  * nazwa i&nbsp;wersja przeglądarki internetowej
  * ustawienia języka
  * strona odsyłająca
  * data i&nbsp;godzina wysłania żądania do&nbsp;serwera
  * IP z&nbsp;którego zostało wysłane żądanie

Adresy IP zbierane przez portal nie mają na&nbsp;celu identyfikacji naszych użytkowników, ani nie są nigdzie ujawniane. Jednakże przy dodawania komentarzy niezgodnych z&nbsp;netykietą, śmiecenia w&nbsp;komentarzach bądź na&nbsp;forum, przewidujemy możliwość zablokowania użytkowania portalu z&nbsp;danego adresu IP, lub w&nbsp;wypadku bardzo nachalnych graczy, nawet z&nbsp;całej puli IP.

#### Gromadzenie danych osobowych

Podczas wypełniania formularza kontaktowego na&nbsp;portalu sysadmin.info.pl lub na&nbsp;jego forum, wymagamy podania przynajmniej adresu E-mail. Podawanie pozostałych danych w&nbsp;profilach, takich jak imię, nazwisko, adres strony internetowej, miejscowość, zawód, zainteresowania, AIM, Jabber, Google Talk – nie jest obowiązkowe, i&nbsp;tylko od&nbsp;Użytkownika zależy czy chce je podać i&nbsp;udostępnić innym.

#### Pliki Cookies

Na&nbsp;portalu sysadmin.info.pl używane są pliki cookies, tzw. ciasteczka – są to&nbsp;małe pliki tekstowe używane do&nbsp;zapisywania pewnych informacji na&nbsp;komputerze Użytkownika. Nie są one niebezpieczne dla Użytkownika ani jego komputera, nie są w&nbsp;nich przechowywane żadne dane osobowe, ich głównym zadaniem jest przetrzymywanie informacji, do&nbsp;których potem strona może się odnieść. Na&nbsp;naszym portalu ciasteczka mają następujące funkcje:

  * przechowywania informacji/preferencji Użytkownika dodającego komentarz – w&nbsp;momencie dodania komentarza na&nbsp;komputerze Użytkownika jest zapisywany plik cookie, który&nbsp;przez niespełna rok przechowuje dane z&nbsp;nickiem, adresem email i&nbsp;stroną internetową Użytkownika – dzięki czemu nie trzeba tych danych wpisywać za&nbsp;każdym razem gdy&nbsp;chce się dodać następny komentarz.
  * przechowywania informacji o&nbsp;Użytkownikach zarejestrowanych/zalogowanych na&nbsp;portalu lub forum – przeglądarka może zapisać plik cookie zawierający login Użytkownika, podwójnie shashowane hasło, oraz&nbsp;klucz autoryzacji, dzięki czemu przy następnej wizycie, użytkownika nie będzie musiał ponownie wpisywać loginu oraz&nbsp;hasła, albo nawet jeśli sesja jeszcze nie wygasła, to&nbsp;nie będzie musiał się ponownie logować – tylko od&nbsp;razu będzie zalogowany na&nbsp;portalu lub forum.

Pliki cookies w&nbsp;nie umożliwiają identyfikacji Użytkowników odwiedzającej witrynę, i&nbsp;nie są w&nbsp;nich zapisywane żadne informacje, które mogły by&nbsp;taką identyfikację umożliwić. Zgodnie z&nbsp;art. 173 Ustawy z&nbsp;dnia 16 lipca 2004 roku Prawo telekomunikacyjne (Dz.&nbsp;U. Nr&nbsp;171, poz. 1800) zapisywanie plików tekstowych na&nbsp;dysku komputera Użytkownika witryny jest dozwolone o&nbsp;ile Użytkownik zostanie o&nbsp;tym poinformowany. Warunkiem działania plików cookies jest ich akceptacja przez przeglądarkę oraz&nbsp;nieusuwanie ich z&nbsp;dysku komputera. Jeżeli Użytkownik nie chce udostępnić plików cookies, powinien taką opcję wyłączyć w&nbsp;przeglądarce nim odwiedzi nasz portal. Nieudostępnienie plików cookies przez przeglądarkę, albo ich skasowanie, możne powodować znaczne utrudnienia lub może nawet całkowicie uniemożliwiać korzystanie z&nbsp;naszego portalu oraz&nbsp;forum – za&nbsp;co nie ponosimy odpowiedzialności. Dlatego zalecamy mieć włączoną obsługę plików cookies.

#### Prawa autorskie

Portal sysadmin.info.pl zawiera artykuły i&nbsp;poradniki napisane przez użytkowników wchodzących w&nbsp;skład Redakcji. Wszystkie treści zawarte na&nbsp;stronach portalu są przedmiotem praw autorskich przysługujących ich twórcy i&nbsp;podlegają ochronie zgodnie z&nbsp;prawem autorskim. Zabrania się kopiowania i&nbsp;udostępniania materiałów z&nbsp;portalu bez zgody od&nbsp;ich autora i&nbsp;zgody od&nbsp;administratora portalu.

Wszelkie materiały zawarte w&nbsp;serwisie portalu sysadmin.info.pl są bezpłatne. Użytkownicy oraz&nbsp;Redakcja portalu nie odpowiadają za&nbsp;jakiekolwiek szkody wynikłe z&nbsp;ich korzystania. Portal sysadmin.info.pl zastrzega sobie możliwość edycji i&nbsp;usuwania wpisów lub komentarzy oraz&nbsp;blokowania użytkowników, którzy&nbsp;naruszają regulamin Portalu, oraz&nbsp;zasady netykiety i&nbsp;powszechnie przyjęte normy kulturalnego zachowania.
